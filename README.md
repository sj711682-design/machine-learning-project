# Email Spam Detection

A beginner Machine Learning project to classify emails as `Spam` or `Not Spam` using `Multinomial Naive Bayes` and `CountVectorizer`.

## 📌 Project Overview
This project was built as part of an Azure ML Guided Project.
The model is trained on a small dataset of email texts and predicts whether a new message is spam `1` or not spam `0`.

## 🛠️ Tech Stack
- **Language**: Python
- **Libraries**: pandas, scikit-learn
- **Model**: MultinomialNB
- **Vectorizer**: CountVectorizer
